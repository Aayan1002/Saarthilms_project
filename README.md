# React Counter App

This project is a simple counter application built using React.js (Class Component).

### Features:
- Increment counter
- Decrement counter
- Reset to zero
- Clean UI

### Author:
Aayan Ansari

### How to Run:

1. Clone the repo:
   git clone https://github.com/YOUR-USERNAME/REPO-NAME.git

2. Install dependencies:
   npm install

3. Start the app:
   npm start
